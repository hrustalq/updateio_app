## О проекте
Game Update Automator - это приложение, разработанное на Tauri + React + TypeScript, которое автоматизирует процесс обновления ваших игр. Больше не нужно вручную запускать Steam или другие лаунчеры для обновления игр - приложение сделает это за вас.

## Возможности
- Автоматическое обновление игр Steam
- Поддержка популярных игр:
  - Dota 2
  - Counter-Strike 2
  - И другие игры Steam
- Уведомления о доступных обновлениях
- Планировщик обновлений

## Требования
- Windows 10/11
- Steam клиент
- Минимум 4 ГБ оперативной памяти
- 100 МБ свободного места на диске

## Установка и запуск
1. Скачайте последнюю версию приложения из раздела [Releases]()
2. Распакуйте архив
3. Запустите `game-update-automator.exe`
4. Следуйте инструкциям по первоначальной настройке
